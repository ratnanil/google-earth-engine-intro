Register for Google Earth Engine (GEE)

Go to code.earthengine.google.com and login either using an existing google account or creating a new one for your academic use. If you have multiple google accounts, we recommend you use the incogniote mode of your browser.

Select Unpaid usage and the Project type Academia & Research and click next

Select Create a new Google Cloud Project. You can keep the default values in Organization, Project-ID and Project Name and click on Next. If you are prompted to that you haven't accepted the Terms and Conditions, click on the link provided, read the Terms and Conditions, and accept these.

Click on Open in Code Editor

If you see this page and can interact with the map (zoom in, zoom out), you are set!
